# 📱 Impacto das Redes Sociais na Saúde Mental

Dashboard interativo desenvolvido para analisar possíveis relações entre comportamento digital, ansiedade, vício digital, qualidade do sono e desempenho acadêmico.

---

# 🚀 Tecnologias utilizadas

- Python
- Pandas
- Plotly
- Streamlit
- Inteligência Artificial (apoio no desenvolvimento)

---

# 🧠 Sobre o Projeto

Esse projeto foi desenvolvido como parte da minha jornada de aprendizado em análise de dados e também como forma de construir meu portfólio em busca da minha primeira oportunidade de estágio na área de tecnologia e dados.

Escolhi esse tema porque o impacto das redes sociais na saúde mental é algo extremamente presente no dia a dia das pessoas — inclusive no meu próprio cotidiano.

Além de praticar tecnologias como Python, Pandas, Plotly e Streamlit, meu objetivo foi transformar dados em informações visuais capazes de gerar reflexão, entendimento e conscientização.

Durante o desenvolvimento do projeto, utilizei Inteligência Artificial como apoio para estudos, organização de ideias, melhorias visuais e otimizações no código. A IA foi utilizada como ferramenta de aprendizado e produtividade durante todo o processo.

---

# 📊 Objetivo do Projeto

O projeto busca analisar como o uso das redes sociais pode impactar diferentes aspectos da saúde mental e do comportamento dos usuários.

Foram analisados indicadores como:

- 📱 Uso de redes sociais
- 😟 Ansiedade
- 🔗 Vício digital
- 😴 Qualidade do sono
- 📚 Desempenho acadêmico

# 📊 Principais análises

- Média de ansiedade por plataforma
- Média de vício digital
- Relação entre sono e ansiedade
- Relação entre vício digital e desempenho acadêmico
- Distribuição de usuários por plataforma
- Correlação entre indicadores
- Insights automáticos baseados nos dados

---

# 💡 Soluções e conscientização

Além da análise dos dados, o projeto também busca conscientizar sobre hábitos digitais mais saudáveis.

Algumas recomendações apresentadas no dashboard incluem:

- Redução do tempo excessivo em redes sociais
- Melhoria da qualidade do sono
- Equilíbrio entre lazer e produtividade
- Uso mais consciente das plataformas digitais
- Criação de hábitos digitais mais saudáveis

---

# 📂 Dataset utilizado

Dataset:
Teen Mental Health Dataset

Fonte:
https://www.kaggle.com/datasets/algozee/teenmentalhealth

O projeto utiliza um dataset público disponibilizado na plataforma Kaggle para fins educacionais e de análise exploratória de dados.

Os dados representam informações relacionadas ao comportamento digital, saúde mental e hábitos digitais de adolescentes, permitindo análises sobre possíveis impactos das redes sociais no cotidiano.

O dataset foi utilizado exclusivamente para fins de aprendizado, desenvolvimento de dashboard e construção de portfólio em análise de dados.
